# TamperTantrum Labs Security Skills

Open-source Claude Code skills for building secure applications by default.

Created and maintained by [TamperTantrum Labs](https://tampertantrum.com) - making application security accessible, human, and empowering.

## Available Skills

| Skill | Description | Install |
|-------|-------------|---------|
| `vast-threat-modeling` | VAST methodology for practical, scalable threat modeling | `npx skills add tampertantrum-labs/security-skills@vast-threat-modeling` |
| `react-secure-coder` | Security-first React/TypeScript patterns | `npx skills add tampertantrum-labs/security-skills@react-secure-coder` |
| `nextjs-security` | Next.js specific security (middleware, API routes, server actions) | `npx skills add tampertantrum-labs/security-skills@nextjs-security` |
| `api-security` | REST/GraphQL API security patterns | `npx skills add tampertantrum-labs/security-skills@api-security` |
| `secure-forms` | Input validation, sanitization, CSRF protection | `npx skills add tampertantrum-labs/security-skills@secure-forms` |
| `auth-patterns` | Authentication done right (JWT, sessions, OAuth) | `npx skills add tampertantrum-labs/security-skills@auth-patterns` |
| `secure-by-default` | Meta-skill: automatically apply security patterns during code generation | `npx skills add tampertantrum-labs/security-skills@secure-by-default` |

## Philosophy

LLMs don't build secure code by default. These skills change that.

Each skill is designed to be **proactive, not reactive** - security patterns are applied during code generation, not discovered during review.
